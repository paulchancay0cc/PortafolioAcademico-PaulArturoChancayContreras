Tema 01
+

Integrales Definidas e Indefinidas

La base de todo el cálculo integral: antiderivadas y el Teorema Fundamental.

Concepto

Una integral indefinida ∫f(x)dx representa a toda la familia de antiderivadas de f(x), es decir, todas las funciones F(x) cuya derivada es f(x); por eso siempre se agrega la constante de integración C. Una integral definida ∫[a→b] f(x)dx en cambio es un número: representa el área neta bajo la curva de f(x) entre x=a y x=b, y se calcula con el Teorema Fundamental del Cálculo.

Fórmula
∫ f(x) dx = F(x) + C ∫[a→b] f(x) dx = F(b) − F(a)

F(x) es cualquier antiderivada de f(x). En la definida, la constante C se cancela al restar.

Ejercicio de ejemplo
Ejercicio Resolver la integral indefinida ∫(3x² + 2x) dx y luego evaluar la integral definida ∫[0→2] (3x² + 2x) dx.
1
Se integra término a término usando la regla de la potencia ∫xⁿdx = xⁿ⁺¹/(n+1).
2
∫3x² dx = x³ y ∫2x dx = x². La antiderivada indefinida es:
3
F(x) = x³ + x² + C
4
Para la integral definida se evalúa F(x) en los límites: F(2) − F(0).
5
F(2) = 2³ + 2² = 8 + 4 = 12
6
F(0) = 0³ + 0² = 0
✔ ∫(3x²+2x)dx = x³ + x² + C | ∫[0→2](3x²+2x)dx = 12

Tema 02
+

Integración por partes

El método basado en la regla del producto para derivadas, aplicado en reversa.

Concepto

Se usa cuando el integrando es un producto de dos funciones de distinta naturaleza (por ejemplo un polinomio por una exponencial, o un logaritmo por x). La clave está en elegir bien u y dv siguiendo el orden LIATE (Logarítmica, Inversa trigonométrica, Algebraica, Trigonométrica, Exponencial) para que la nueva integral ∫v du sea más sencilla que la original.

Fórmula
∫ u dv = u·v − ∫ v du

Se deriva u y se integra dv. Si la nueva integral sigue siendo compleja, se repite el proceso.

Ejercicio de ejemplo
Ejercicio Resolver ∫ x·eˣ dx usando integración por partes.
1
Se elige u = x (se deriva) y dv = eˣ dx (se integra), siguiendo el orden LIATE.
2
Entonces du = dx y v = eˣ.
3
Se aplica la fórmula ∫u dv = uv − ∫v du:
4
∫x·eˣ dx = x·eˣ − ∫eˣ dx
5
Se resuelve la integral restante: ∫eˣ dx = eˣ
✔ ∫ x·eˣ dx = x·eˣ − eˣ + C

Tema 03
+

Integral por cambio de variable

También llamado método de sustitución: simplifica la integral con una nueva variable u.

Concepto

Consiste en identificar una parte del integrando como una nueva variable u = g(x), de modo que du = g'(x)dx aparezca (o pueda hacerse aparecer) en la integral. Esto transforma una integral complicada en una integral básica en términos de u, que luego se devuelve a la variable original.

Fórmula
Sea u = g(x), du = g'(x) dx ∫ f(g(x))·g'(x) dx = ∫ f(u) du

Es el método más usado como primer intento antes de recurrir a partes o sustitución trigonométrica.

Ejercicio de ejemplo
Ejercicio Resolver ∫ 2x·(x²+1)⁴ dx mediante cambio de variable.
1
Se identifica u = x² + 1, porque su derivada aparece en el integrando.
2
Se deriva: du = 2x dx (¡coincide exactamente con lo que sobra en la integral!).
3
Se sustituye: ∫2x(x²+1)⁴ dx = ∫u⁴ du
4
Se integra en términos de u: ∫u⁴ du = u⁵/5 + C
5
Se regresa a la variable original reemplazando u = x²+1.
✔ ∫ 2x·(x²+1)⁴ dx = (x²+1)⁵ / 5 + C

Tema 04
+

Integrales trigonométricas (tipo 1, 2, 3, 4)

Potencias y productos de funciones trigonométricas: seno, coseno, tangente y secante.

Concepto

Se clasifican según la combinación de funciones y la paridad de sus exponentes. Tipo 1: potencias impares de seno o coseno (se separa un factor y se usa la identidad pitagórica). Tipo 2: potencias pares de seno y coseno (se usan identidades de ángulo doble). Tipo 3: potencias de tangente y secante. Tipo 4: productos de seno y coseno con ángulos distintos (se usan fórmulas de producto a suma).

Fórmula
sin²x + cos²x = 1 tan²x + 1 = sec²x sin²x = (1−cos2x)/2 cos²x = (1+cos2x)/2

Identificar primero si las potencias son pares o impares determina qué identidad conviene aplicar.

Ejercicio de ejemplo
Ejercicio Resolver ∫ sin³x dx (potencia impar de seno).
1
Como la potencia es impar, se separa un factor de seno: sin³x = sin²x · sinx.
2
Se usa la identidad pitagórica: sin²x = 1 − cos²x.
3
La integral queda: ∫(1 − cos²x)·sinx dx
4
Se sustituye u = cosx, du = −sinx dx, entonces sinx dx = −du.
5
∫(1−u²)(−du) = −∫(1−u²)du = −u + u³/3 + C
6
Se regresa a la variable original reemplazando u = cosx.
✔ ∫ sin³x dx = −cosx + cos³x/3 + C

Tema 05
+

Integral por sustitución trigonométrica

Para integrandos con raíces del tipo √(a²−x²), √(a²+x²) o √(x²−a²).

Concepto

Cuando aparece una raíz cuadrada de una suma o diferencia de cuadrados, se sustituye x por una función trigonométrica (seno, tangente o secante) para eliminar la raíz usando identidades pitagóricas, formando un triángulo de referencia para regresar a la variable original al final.

Fórmula
√(a²−x²) → x = a·sen θ √(a²+x²) → x = a·tan θ √(x²−a²) → x = a·sec θ

Al final se usa un triángulo rectángulo para expresar θ de nuevo en función de x.

Ejercicio de ejemplo
Ejercicio Resolver ∫ dx / √(4 − x²) mediante sustitución trigonométrica.
1
La raíz tiene la forma √(a²−x²) con a = 2, así que se sustituye x = 2·senθ.
2
Entonces dx = 2·cosθ dθ y √(4−x²) = √(4−4sen²θ) = 2·cosθ.
3
Se sustituye en la integral: ∫ (2·cosθ dθ) / (2·cosθ) = ∫ dθ
4
Se integra directamente: ∫dθ = θ + C
5
Se regresa a x usando θ = arcsen(x/2).
✔ ∫ dx / √(4−x²) = arcsen(x/2) + C

Segundo Parcial

Aplicaciones de la integral

Una vez dominadas las técnicas de integración, se aplican para resolver problemas geométricos y físicos: áreas, volúmenes, longitudes de curva y cinemática.

Tema 01
+

Integración de funciones racionales (fracciones parciales)

Caso I: factores lineales no repetidos · Caso II: factores lineales repetidos.

Concepto

Cuando el integrando es una fracción de polinomios, se descompone en una suma de fracciones más simples antes de integrar. Caso I: si el denominador se factoriza en factores lineales distintos, a cada uno le corresponde una fracción con una constante en el numerador. Caso II: si un factor lineal se repite k veces, le corresponden k fracciones, una por cada potencia del factor de 1 hasta k.

Fórmula
Caso I: P(x)/[(x−a)(x−b)] = A/(x−a) + B/(x−b) Caso II: P(x)/(x−a)^k = A₁/(x−a) + A₂/(x−a)² + … + Aₖ/(x−a)^k

El grado del numerador debe ser menor que el del denominador; si no, primero se hace división de polinomios.

Ejercicio de ejemplo
Ejercicio Resolver ∫ (3x+5) / [(x−1)(x+2)] dx (Caso I: factores lineales no repetidos).
1
Se plantea la descomposición: (3x+5)/[(x−1)(x+2)] = A/(x−1) + B/(x+2)
2
Se multiplica ambos lados por (x−1)(x+2): 3x+5 = A(x+2) + B(x−1)
3
Se evalúa en x = 1: 8 = A(3) → A = 8/3
4
Se evalúa en x = −2: −1 = B(−3) → B = 1/3
5
Se integra cada fracción por separado: ∫A/(x−1)dx + ∫B/(x+2)dx
✔ ∫(3x+5)/[(x−1)(x+2)] dx = (8/3)ln|x−1| + (1/3)ln|x+2| + C

Tema 02
+

Área bajo la curva

Sus fórmulas, gráficas y variantes: área entre el eje X y una función, y área entre dos curvas.

Concepto

El área de la región R limitada por la gráfica de f(x), el eje X y las rectas x=a, x=b se calcula con la integral definida de f en [a,b]. Cuando f(x) es negativa en parte del intervalo, se usa el valor absoluto para que el área no salga negativa. Si la región está limitada por dos curvas (RL: región limitada), se integra la diferencia entre la curva superior y la inferior.

Gráfica
Representación gráfica del tema · Área bajo la curva
Fórmula
Área bajo la curva: A = ∫[a→b] |f(x)| dx Área entre dos curvas (f arriba de g): A = ∫[a→b] [f(x) − g(x)] dx

Siempre conviene graficar la región y hallar los puntos de intersección antes de plantear la integral.

Ejercicio de ejemplo
Ejercicio Calcular el área bajo la curva f(x) = x² entre x = 0 y x = 2.
1
Se plantea la integral definida: A = ∫[0→2] x² dx
2
Se halla la antiderivada: ∫x² dx = x³/3
3
Se evalúa en los límites: A = [x³/3] desde 0 hasta 2
4
A = (2³/3) − (0³/3) = 8/3 − 0
✔ Área = 8/3 unidades² ≈ 2.67 u²

Tema 03
+

Volúmenes · Sólidos de revolución

Sus fórmulas y gráficas: método de discos y método de arandelas.

Concepto

Al girar una región plana alrededor de un eje se genera un sólido de revolución. Si la región toca el eje de giro se usa el método de discos (secciones circulares macizas); si hay un hueco entre la región y el eje se usa el método de arandelas (secciones en forma de anillo), restando el volumen del radio interior al del radio exterior.

Gráfica
Representación gráfica del tema · Volúmenes · Sólidos de revolución
Fórmula
Método de discos (eje X): V = π ∫[a→b] [f(x)]² dx Método de arandelas: V = π ∫[a→b] [R(x)² − r(x)²] dx

R(x) es el radio exterior (curva más lejana al eje) y r(x) el radio interior (curva más cercana al eje).

Ejercicio de ejemplo
Ejercicio La región bajo la curva y = √x, entre x = 0 y x = 4, gira alrededor del eje X. Calcular el volumen del sólido generado (método de discos).
1
Se identifica el radio del disco: R(x) = f(x) = √x
2
Se plantea la fórmula del método de discos: V = π ∫[0→4] [√x]² dx
3
Se simplifica el integrando: [√x]² = x, entonces V = π ∫[0→4] x dx
4
Se integra: ∫x dx = x²/2
5
Se evalúa entre 0 y 4: V = π[(4²/2) − (0²/2)] = π(8 − 0)
✔ Volumen = 8π unidades³ ≈ 25.13 u³

Tema 04
+

Longitud de curva

Sus variantes y fórmulas: forma explícita y forma paramétrica.

Concepto

La longitud de arco de una curva se obtiene sumando infinitos segmentos infinitesimales sobre la curva. Si la curva está dada como y=f(x), se usa la derivada dy/dx dentro de la fórmula. Si está dada en forma paramétrica x(t), y(t), se usan las derivadas respecto al parámetro t.

Fórmula
Forma explícita: L = ∫[a→b] √(1 + (dy/dx)²) dx Forma paramétrica: L = ∫[a→b] √((dx/dt)² + (dy/dt)²) dt

En coordenadas polares se usa una fórmula análoga con r(θ) y dr/dθ.

Ejercicio de ejemplo
Ejercicio Calcular la longitud de la curva f(x) = (2/3)x^(3/2) entre x = 0 y x = 3.
1
Se deriva la función: dy/dx = x^(1/2)
2
Se eleva al cuadrado: (dy/dx)² = x
3
Se sustituye en la fórmula: L = ∫[0→3] √(1 + x) dx
4
Se integra usando u = 1+x, du = dx: ∫√u du = (2/3)u^(3/2)
5
Se evalúa entre los límites: L = (2/3)[(1+x)^(3/2)] desde 0 hasta 3
6
L = (2/3)[(4)^(3/2) − (1)^(3/2)] = (2/3)[8 − 1]
✔ Longitud = 14/3 unidades ≈ 4.67 u

Tema 05
+

Cinemática

Posición, velocidad y aceleración obtenidas mediante integración.

Concepto

En cinemática, la velocidad es la antiderivada de la aceleración y la posición es la antiderivada de la velocidad. A partir de la función de aceleración a(t) y las condiciones iniciales (posición y velocidad en t=0), se puede reconstruir por completo el movimiento de una partícula usando integrales definidas.

Fórmula
v(t) = v₀ + ∫[0→t] a(τ) dτ x(t) = x₀ + ∫[0→t] v(τ) dτ

Las condiciones iniciales x₀ y v₀ son indispensables para determinar las constantes de integración.

Ejercicio de ejemplo
Ejercicio Una partícula tiene aceleración a(t) = 6t. Si v(0) = 2 y x(0) = 0, hallar v(t), x(t) y sus valores en t = 2.
1
Se integra la aceleración para obtener la velocidad: v(t) = ∫6t dt = 3t² + C₁
2
Se usa la condición inicial v(0)=2: 3(0)²+C₁=2 → C₁=2, entonces v(t) = 3t² + 2
3
Se integra la velocidad para obtener la posición: x(t) = ∫(3t²+2) dt = t³ + 2t + C₂
4
Se usa la condición inicial x(0)=0: (0)³+2(0)+C₂=0 → C₂=0, entonces x(t) = t³ + 2t
5
Se evalúan en t=2: v(2) = 3(4)+2 = 14 · x(2) = 8+4 = 12
✔ v(t) = 3t² + 2 → v(2) = 14 | x(t) = t³ + 2t → x(2) = 12

Sobre la materia

Concepto de Cálculo Integral y aprendizaje de la materia

El Cálculo Integral es la rama de las matemáticas que estudia la acumulación de cantidades y el cálculo del área bajo una curva. Es el proceso inverso a la derivada: mientras la derivada mide el cambio instantáneo de una función, la integral suma infinitos cambios infinitesimales para reconstruir el total. Su estudio combina técnicas algebraicas de integración (sustitución, partes, fracciones parciales, sustitución trigonométrica) con aplicaciones geométricas y físicas (áreas, volúmenes, longitudes de curva y cinemática).
